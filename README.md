# README
  
Tested on ubuntu 20.04

## get repo
```
git clone --recurse-submodules  git@github.com:patrickhallinan/hello-proxygen
```

## build proxygen

**From the root directory**

```
cd proxygen/proxygen
./build.sh
```

## build hello proxygen

**From the root directory**

```
mkdir build
cd build
cmake ..
make
```
